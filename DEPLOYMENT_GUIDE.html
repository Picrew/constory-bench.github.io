📤 部署到 GitHub Pages

1

创建 GitHub 仓库

• 登录 GitHub

• 点击 "New repository"

• 名称: long-story-bench

• 设置为 Public（必须公开）

• 不要初始化任何文件

2

推送代码到 GitHub

cd "d:\ms\StoryVerify\generate_story\benchmark\website\long-story-bench" git init git add . git commit -m "Initial commit: Long-Story Bench website" # 替换 YOUR-USERNAME 为你的 GitHub 用户名 git remote add origin https://github.com/YOUR-USERNAME/long-story-bench.git git branch -M main git push -u origin main
3

启用 GitHub Pages

1. 进入你的 GitHub 仓库

2. 点击 Settings （设置）

3. 左侧菜单找到 Pages

4. Source 选择:

• Branch: main

• Folder: / (root)

5. 点击 Save

4

访问你的网站

等待 1-2 分钟后，你的网站将在以下地址可用:

https://YOUR-USERNAME.github.io/long-story-bench/
⚠️ 记得将 YOUR-USERNAME 替换为你的 GitHub 用户名
